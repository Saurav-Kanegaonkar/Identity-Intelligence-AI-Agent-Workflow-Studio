# Data Sources

Synthetic source-style data for identity intelligence AI agents, workflow automation, contact-data research, internal training, and agent quality review.

- `entities.csv`: 36 agent workflow records.
- `daily_metrics.csv`: 5,040 agent workflow-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
